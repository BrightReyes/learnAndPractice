# 🐛 PHP Debugging Practice — Easy Level

## How to Play

1. Open **`challenge.php`** — it contains **10 bugs**
2. Run it with `php challenge.php` to see which tests fail
3. Read the comments above each section to understand what the code *should* do
4. Find and fix each bug
5. Re-run the file after each fix to track your progress

## Goal

Get all 10 tests to pass:

```
🎉 All 10 tests passed! Great debugging!
```
